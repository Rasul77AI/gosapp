
«Куда обратиться?» — цифровой навигатор по госорганам

Пилотный проект: пользователь описывает проблему, система определяет
компетентный орган, контакты, документы и сроки. Собирает статистику
самых частых обращений — полезно для прокуратуры/акимата.

## Структура

- data.py — база категорий (легко расширять: добавляй новые словари в CATEGORIES)
- matcher.py — классификатор по ключевым словам (офлайн, без API)
- app.py — веб-интерфейс на Flask + статистика (SQLite)
- bot.py — Telegram-бот на той же логике

## Запуск веб-версии
pip install flask
python app.py
Открыть http://localhost:5000

## Запуск Telegram-бота
pip install python-telegram-bot==21.*
export TELEGRAM_TOKEN=ваш_токен_от_BotFather
python bot.py

## Как добавить новую категорию

Открой data.py и добавь в список CATEGORIES новый словарь по образцу:
{
    "id": "new_category",
    "title": "Название проблемы",
    "keywords": ["ключевая фраза 1", "ключевая фраза 2"],
    "organ": "Компетентный орган",
    "escalation": ["Куда обращаться при бездействии"],
    "contacts": "Телефон/адрес",
    "link": "Ссылка на подачу обращения",
    "documents": ["Документ 1", "Документ 2"],
    "deadline": "Срок рассмотрения",
    "law": "Нормативная база",
},
